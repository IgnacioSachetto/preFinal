Tecnologías

Algunas de las tecnologías y herramientas que utilizó en los proyectos.

GENERAL

Power BI

Herramienta de visualización de datos.

SQL Server

Sistema de gestión de base de datos relacional, desarrollado por la empresa Microsoft.

HTML

HyperText Markup Language,lenguaje de marcado para la elaboración de páginas web.

Wordpress

Sistema de gestión de contenidos (CMS) que permite crear y mantener un blog u otro tipo de web.

MySQL

MySQL es el sistema de gestión de bases de datos relacional más extendido en la actualidad al estar basada en código abierto.

PHP

Lenguaje de programación de uso general que se adapta especialmente al desarrollo web

Visual Studio

Entorno de desarrollo integrado para Windows y macOS.

JAVA

Tecnología que se usa para el desarrollo de aplicaciones que convierten a la Web en un elemento más interesante y útil.
